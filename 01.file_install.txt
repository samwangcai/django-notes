Install Django
    1. download file --  https://www.djangoproject.com/download/
    1. unzip
    2. python setup.py install

Install Setuptools 
    1. download ez_setyp.py -- https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py
    2. run python ez_setup.py
    
Install Pip 
    1. download file --  https://pypi.python.org/pypi/pip
    2. unzip to folder
    3. python setup install
    4. add evn path (when install will see). eg: C:\Users\wangshiwang\AppData\Roaming\Python\Python27\site-packages
